Product Performance Dashboard
This project is an end‑to‑end Power BI dashboard analysing revenue, profit, margin, and sales performance across a small product catalogue. The goal was to take messy spreadsheet data and turn it into something clear, visual, and useful for decision‑making.

What I Built

KPI cards for revenue, profit, and average revenue
Revenue, profit, and margin comparisons
Scatter plot showing the relationship between units sold and profit
Insights table with rankings and product metrics
Clean, business‑focused layout

Tools Used

Power BI — data modelling, DAX, visualisation
SQL — data cleaning and transformation
Excel — initial data preparation

Key Insights

High‑margin products are not always the highest revenue drivers
Profitability varies significantly across product categories
Units sold and profit show a clear positive relationship
Several products underperform despite strong margins
